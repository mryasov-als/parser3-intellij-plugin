reverse. Обратный порядок элементов 

^массив.reverse[]

Метод возвращает новый массив, в котором инициализированные элементы идут в порядке, обратном порядку их расположения в текущем массиве.

Пример
$man[Вася;22;m]
$man[^man.reverse[]]
^man.foreach[key;value]{
        $key=$value
}[<br />]

Выведет на экран:
0=m
1=22
2=Вася
