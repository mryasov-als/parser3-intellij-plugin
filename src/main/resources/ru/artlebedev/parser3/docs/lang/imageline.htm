line. Рисование линии на изображении 

^картинка.line(x0;y0;x1;y1;цвет)    

Метод рисует на изображении линию заданного цвета от точки с координатами (х0:y0) до точки (x1:y1).

Пример
$square[^image::create(100;100;0x000000)]
^square.line(0;0;100;100;0xFFFFFF)
$response:body[^square.gif[]]

В браузере будет выведен черный квадрат размером 100 х 100 пикселей, перечеркнутый по диагонали белой линией.
