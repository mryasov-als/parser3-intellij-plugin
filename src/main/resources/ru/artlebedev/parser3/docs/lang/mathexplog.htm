exp, log, log10. Логарифмические функции 

Методы вычисляют значения логарифмических функций от заданного числа.

^math:exp(число)
- Экспонента числа по основанию e
^math:log(число)
- Натуральный логарифм числа
^math:log10(число)
- Десятичный логарифм числа


Примечание: логарифм по произвольному основанию base вычисляется как log(число)/log(base).
