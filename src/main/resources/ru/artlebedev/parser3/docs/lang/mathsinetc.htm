sin, asin, cos, acos, tan, atan, atan2. Тригонометрические функции 

^math:sin(радианы)
- синус
^math:asin(число)
- арксинус
^math:cos(радианы)
- косинус
^math:acos(число)
- арккосинус
^math:tan(радианы)
- тангенс
^math:atan(число)
- арктангенс
^math:atan2(число;число)
- четырехквадрантный арктангенс  [3.5.0]


Методы вычисляют значения тригонометрических функций от заданного числа.

Пример
^math:cos(^math:radians(180)) - получаем -1 (cos p = -1).
