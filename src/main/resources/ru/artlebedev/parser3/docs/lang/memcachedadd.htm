add. Добавление записи 

^memcached.add[ключ;строка]

Если на сервере уже есть запись с указанным ключом, то метод ничего не делает и возвращает false.
Если такой записи нет - метод сохраняет ее на сервере и возвращает true.

Обычно для записи данных нужно применять конструкцию $memcached.[$key][$value]
