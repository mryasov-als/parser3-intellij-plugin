Запись 

$memcached.ключ[значение]
$memcached.ключ[
   $.value[значение]
   ...необязательный модификатор...
]

Сохраняет на сервер ассоциацию между ключом и строкой.

Необязательный модификатор:
$.expires(число секунд) - задает число секунд, на которое сохраняется пара «ключ / строка», 0 = навсегда.
