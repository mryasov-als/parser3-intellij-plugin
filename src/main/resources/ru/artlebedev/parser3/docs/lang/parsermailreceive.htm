Использование Parser в качестве обработчика сообщений по электронной почте 

Рекомендуется настроить свою почтовую систему таким образом, чтобы при приходе сообщения на определенный адрес или адреса запускался некоторый скрипт на Parser (см. «Использование Parser в качестве интерпретатора скриптов»), задав в командной строке интерпретатора ключ -m

По получении письма система запустит созданный скрипт, в котором будет доступно поле
$mail:received
с принятым письмом.

Авторам почтовых рассылок (подписка на новости и прочее): при подписке пользователя на рассылку настоятельно рекомендуется направить ему письмо с адреса xxx-confirm@вашдомен.ru с просьбой подтвердить его желание, просто нажав кнопку «Ответить» в его почтовом клиенте. Следует организовать прием его подтверждения и направлять письма рассылки только после подтверждения. Также рекомендуется принимать сообщения от почтовых роботов об ошибках доставки писем. Неправильные адреса следует удалять из баз данных.
