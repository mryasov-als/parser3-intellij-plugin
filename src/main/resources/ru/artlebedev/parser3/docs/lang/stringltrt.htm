left, right. Подстрока слева и справа 

^строка.left(N)  
^строка.right(N)   

Методы возвращают N первых или последних символов строки соответственно. Если длина строки меньше N, то возвращается вся строка. При вызове ^строка.left(-1) возвращается вся строка.   [3.4.4]

Пример
$str[О, сколько нам открытий чудных!…]
^str.left(10) ^str.right(10)

На экран будет выведено: О, сколько чудных!…
