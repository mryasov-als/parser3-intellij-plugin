file. Преобразование документа в объект класса file 

^документ.file[Параметры_преобразования_в_текст]

Метод преобразует документ в тип file. Возможно задание параметров_преобразования в текст.
По умолчанию создается XML-представление документа с заголовком <?xml … ?> (можно отключить вывод заголовка, задав соответствующий параметр).

Метод понимает опцию $.file[имя файла], с помощью которой можно задать имя создаваемому объекту типа file.   [3.4.2]


Пример
$document[^xdoc::create{<?xml version="1.0" encoding="windows-1251" ?>
<document>
строка1<br/>
строка2<br/>
</document>}]

$response:body[^document.file[]]
