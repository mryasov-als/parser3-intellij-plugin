save. Сохранение документа в файл 

^документ.save[путь]
^документ.save[путь;Параметры_преобразования_в_текст]

Метод сохраняет документ в текстовый файл. Возможно задание параметров_преобразования в текст. По умолчанию создается XML-представление документа с заголовком <?xml … ?> (можно отключить вывод заголовка, задав соответствующий параметр).

Путь - путь к файлу.


Пример
$document[^xdoc::create{<?xml version="1.0" encoding="windows-1251" ?>
<document>
строка1<br/>
строка2<br/>
</document>}]

^document.save[saved.xml]
